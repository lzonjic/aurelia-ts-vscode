# Setup
## 1. Clone repo
`git clone address dir`

## 2. Change working directory
`cd dir`

## 3. Run VS Code from
`code .`

## 4. Installing NPM packages
`npm install`

## 5. Install TypeScript Definitions
`tsd install`

## 6. Install JSPM packages
`jspm install`

## 7. Run build, watch and nodemon
`gulp`
